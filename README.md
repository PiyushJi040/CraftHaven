# CraftHaven

CraftHaven is a web-based platform for crafting enthusiasts to browse, sell, and purchase handmade crafts and products. It features a user-friendly interface for shopping, selling, and managing accounts.

## Features

- **User Authentication**: Login and signup functionality for users.
- **Product Browsing**: Explore a wide range of craft products with categories and images.
- **Selling Platform**: Sellers can list their handmade items.
- **Shopping Cart**: Add products to cart and proceed to checkout.
- **Payment Integration**: Secure payment processing.
- **Responsive Design**: Optimized for desktop and mobile devices.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Clone the repository:
   ```
   git clone https://github.com/HUNT136/CraftHaven.git
   ```

2. Navigate to the project directory:
   ```
   cd CraftHaven
   ```

3. Open `index.html` in your web browser to start exploring the site.

## Project Structure

- `index.html`: Home page
- `login.html`: User login page
- `signup.html`: User registration page
- `sell.html`: Seller dashboard
- `checkout.html`: Checkout process
- `css/`: Stylesheets
- `js/`: JavaScript files
- `images/`: Product and UI images
